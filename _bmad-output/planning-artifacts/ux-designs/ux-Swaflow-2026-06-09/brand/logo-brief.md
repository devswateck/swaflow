# Brief de logo - SWAFLOW

## Objetivo

Crear el logo oficial de la aplicacion SWAFLOW, alineado a la identidad Swa Tech: tecnologia tactica, ventas conversacionales, IA operativa, WhatsApp, confianza comercial y lectura SaaS profesional.

## Concepto v1 generado

Archivo de referencia:

- `brand/swaflow-logo-concept-v1.png`

La exploracion incluye:

- Wordmark `SWAFLOW`.
- Icono compacto con monograma `S`.
- Lockup combinado icono + wordmark.
- Paleta dark/magenta/violeta alineada con `DESIGN.md`.
- Concepto verbal: precision operativa + innovacion tactica.

## Direccion recomendada

Usar el **lockup combinado** como base:

- Icono compacto para favicon, app icon, sidebar colapsado y loading.
- Wordmark `SWAFLOW` para login, sidebar expandido, documentos y material comercial.
- Acento magenta como detalle de flujo, no como decoracion masiva.

## Ajustes antes de produccion

- Redibujar en vector. El PNG generado es referencia visual, no arte final.
- Revisar legibilidad de `W` y del final `FLOW` en tamanos pequenos.
- Confirmar si se conserva tagline `TACTICAL INNOVATION` dentro del logo o solo para materiales de marca. Para la app, el tagline no debe aparecer en el sidebar.
- Ajustar el monograma para que no dependa de brillo/sombra.
- Crear version monocromatica para fondos claros/oscuros.
- Crear version reducida para 32px y favicon.

## Assets finales requeridos

- `swaflow-logo.svg` - lockup horizontal.
- `swaflow-mark.svg` - icono/monograma.
- `swaflow-logo-dark.svg` - si el lockup requiere variante para dark mode.
- `swaflow-logo-light.svg` - si el lockup requiere variante para light mode.
- `favicon.svg` o `favicon.ico`.
- `apple-touch-icon.png` / `pwa-192.png` / `pwa-512.png` si se decide soporte PWA.

## Uso en frontend

- Login: lockup horizontal.
- Sidebar desktop: lockup compacto `SWAFLOW`.
- Sidebar colapsado/mobile: monograma.
- Loading screen: monograma + texto "Validando sesion".
- Favicon: monograma simplificado.

## Reglas

- No usar verde.
- No usar el logo del cliente/tenant como identidad principal de la plataforma.
- No depender de efectos 3D, glow fuerte o sombras para reconocer el logo.
- Mantener `SWAFLOW` como texto visible de producto.
