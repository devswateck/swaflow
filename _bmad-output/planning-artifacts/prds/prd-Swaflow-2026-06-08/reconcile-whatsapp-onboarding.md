# Reconcile: WhatsApp Onboarding Research

## Resultado

La investigacion se conserva como soporte para V2, no como alcance V1.

## Capturado

- V1 mantiene configuracion tecnica actual de WhatsApp Cloud API.
- V2 contempla autenticacion/popup Meta Embedded Signup.
- PRD conserva requisitos de credenciales cifradas, webhook, prueba de cuenta y recepcion/envio.

## Gaps

- Para V2 hara falta especificacion dedicada de permisos Meta, revision de app, OAuth/Embedded Signup y flujo de activos WABA.
